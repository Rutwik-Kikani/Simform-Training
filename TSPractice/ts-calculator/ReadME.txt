/**Setup Instruction**/
goto ts-calculator using cd , or open ts-calculator in visual studio code
open terminal and 
1.run the following command 'npm install' (which will install lite-server)
2.run npm start (which will run the lite-server)
open other terminal
1.run tsc --watch

/**Instruction for use**/
1. keyboard support: shown keys only, other than that key press will get alert.
2. for clearing screen press "esc" key
3. for answer press "enter" key.
4. if display say "Error" means the math expression you enter is invalid, wait for 1s
5. Suggetion: clear display before goto next expression
